# Eazy HWPX 테스터 가이드
> 작성일: 2026-04-12
> 목적: AI 매핑 품질 검증 — 테스터가 실제로 툴을 써보고 결과를 기록하는 절차

---

## 준비물

1. **테스트 양식** — 이 폴더 안 `.hwp` 파일 7개
2. **입력 데이터** — `test-inputs/` 폴더 안 `.txt` 파일 7개 (복사해서 붙여넣기용)
3. **결과 기록지** — 이 문서 맨 아래 `결과 기록` 섹션

툴 주소: https://eazyhwpx.com  
테스트 계정: (테스터 본인 회원가입)

---

## 테스트 방법 (공통 절차)

```
1. 툴 접속 → 로그인
2. "AI 매핑" 탭 선택
3. 양식 업로드 (.hwp 파일)
4. 입력 데이터 텍스트 붙여넣기 (txt 파일 내용 전체 복사)
5. 생성 버튼 클릭
6. 결과 다운로드 → HWP에서 열기
7. 아래 채점 기준으로 등급 판정 후 기록
```

---

## 채점 기준

| 등급 | 정확도 | 판정 |
|-----|-------|-----|
| **A** | 90%+ | 즉시 상용화 가능 |
| **B** | 75~89% | 거의 다 됨, 몇 개 확인 필요 |
| **C** | 50~74% | 반은 맞고 반은 틀림 |
| **D** | 50% 미만 | 수동 입력이 나음 |

**필드별 채점 방법**: 양식에 있는 전체 필드 수를 세고, 올바르게 채워진 필드 수를 나눔  
예) 전체 20개 필드 중 17개 정확 → 85% → B등급

**오류 유형 분류**
- `위치 오류`: 맞는 값인데 엉뚱한 칸에 들어감
- `혼동 오류`: 비슷한 두 필드를 서로 뒤바꿈 (예: 공급자↔공급받는자)
- `누락`: 입력 데이터에 있는데 양식에 안 채워짐
- `할루시네이션`: 입력 데이터에 없는 내용이 생성됨
- `포맷 오류`: 값은 맞지만 형식이 다름 (예: 날짜 "20260412" → 기대값 "2026년 4월 12일")
